# NCOS Dissolved Oxygen and Water Quality Project

## Project overview

This project examines dissolved oxygen (DO) patterns in NCOS water quality monitoring data. We are interested in how DO differs across sites and whether DO changes across sampling elevations within each site. These patterns may help us understand whether some sites show stronger vertical differences in water quality than others.

The project currently focuses on three main NCOS monitoring sites:

- East Channel
- Phelps Bridge
- Venoco Bridge

## Research questions

1. How does dissolved oxygen differ across NCOS monitoring sites?
2. How does dissolved oxygen vary across elevation codes within each site?
3. Do differences in dissolved oxygen across elevation codes suggest possible stratification?
